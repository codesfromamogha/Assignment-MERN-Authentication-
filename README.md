# MERN Authentication System

## Features
- Local signup & login (JWT)
- Google OAuth 2.0
- Facebook OAuth
- Twitter OAuth
- Protected routes
- MongoDB user storage

## Tech Stack
- MongoDB
- Express
- React
- Node.js
- Passport.js
- JWT

## Setup

### Backend
```bash
cd server
npm install
npm run dev

###Frontend
```bash
cd client
npm install
npm start
